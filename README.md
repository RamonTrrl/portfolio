## 📋 Descripción

Este portafolio es mi carta de presentación en el mundo digital. Está diseñado para reflejar mi identidad como **"obrero digital"** —una persona apasionada por la tecnología que construye soluciones creativas con código.

Incluye secciones sobre:
- **Acerca de mí**: mi enfoque y filosofía de trabajo.
- **Trabajo / Proyectos**: una galería con algunos de mis desarrollos.
- **Contacto**: vías para conectar conmigo.

## 🛠️ Tecnologías utilizadas

- **HTML5**: estructura semántica y accesible.
- **CSS3**: estilos personalizados con enfoque responsive y uso de Flexbox/Grid.
- **GitHub Pages**: hosting gratuito y despliegue continuo.

## ✨ Características

- Diseño responsive (se adapta a móviles, tablets y escritorio).
- Navegación fluida entre secciones.
- Presentación visual de proyectos con tarjetas (Cards).
- Iconos y enlaces a redes sociales y blog.
